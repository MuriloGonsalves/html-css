Principais formatações

Negrito/ Destaque

Nessa frase temos um termo em negrito usando a tag B (não semantica).

Nesssa frase tenho um termo em destaque usando a tag strong.

Italico / Enfase

Nesta frase,temos um termo em Italico usando a tag I (não semantica)

Nesta frase,temos um termo em ênfase usando a tag EM (semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo,temos outro texto marcado no final.

Texto menor

Texto pequeno nesse parágrafo.

Para colocar um texto grande use css,ja para colocar menor use a tag small.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido,mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
